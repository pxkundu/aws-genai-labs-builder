# Claude Code Exercises

This directory contains hands-on exercises for learning Claude Code with AWS.

## Exercise List

### Exercise 1: Basic Code Generation
**File**: `exercise_1_basic_code_generation.py`

Generate a simple Python function using Claude Code. Learn the basics of code generation.

**Learning Objectives**:
- Create ClaudeCodeClient instance
- Generate code from natural language prompts
- Understand code generation parameters

### Exercise 2: API Generation
**File**: `exercise_2_api_generation.py`

Generate a REST API endpoint using Claude Code. Learn how to generate API code.

**Learning Objectives**:
- Generate API endpoints with Claude Code
- Understand prompt engineering for APIs
- Save and use generated code

### Exercise 3: Infrastructure as Code
**File**: `exercise_3_infrastructure_generation.py`

Generate AWS infrastructure code using Claude Code. Learn IaC generation.

**Learning Objectives**:
- Generate AWS CDK/Terraform code
- Understand infrastructure requirements
- Deploy generated infrastructure

### Exercise 4: Code Refactoring
**File**: `exercise_4_code_refactoring.py`

Refactor existing code using Claude Code. Learn code improvement techniques.

**Learning Objectives**:
- Refactor code with Claude Code
- Understand refactoring instructions
- Improve code quality

### Exercise 5: Test Generation
**File**: `exercise_5_test_generation.py`

Generate unit tests using Claude Code. Learn test generation.

**Learning Objectives**:
- Generate unit tests with Claude Code
- Understand test coverage requirements
- Write comprehensive tests

## How to Complete Exercises

1. Read the exercise file
2. Follow the TODO comments
3. Complete the implementation
4. Test your solution
5. Compare with solutions (if available)

## Prerequisites

- Completed Module 1: Environment Setup
- Completed Module 2: Claude Code Basics
- AWS credentials configured
- Bedrock access with Claude models
